# nature-skills
# AGENTS.md

This repository contains Codex skills under `.agents/skills`.

When the user asks for academic paper reading, Chinese Markdown readers, journal-club PPTs, Nature-style polishing, citation checking, reviewer responses, or publication-quality figures, first check whether a matching `nature-*` skill exists.

Use these skills when relevant:

- `nature-reader`: paper reading, bilingual Markdown reader, figure-aware summaries.
- `nature-paper2ppt`: convert papers into journal-club PPT content.
- `nature-polishing`: Nature-style academic polishing.
- `nature-citation`: citation checking and reference support.
- `nature-response`: reviewer response drafting.
- `nature-writing`: academic writing support.
- `nature-data`: data-related scientific writing or analysis support.
- `nature-figure`: publication-style scientific figure workflows.
- `nature-academic-search`: academic search and literature retrieval workflows.

When using a skill, read the skill folder, not only `SKILL.md`, because references, scripts, and assets may be required.
